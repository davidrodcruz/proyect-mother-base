# Product

## Register

brand

## Users

Tech recruiters, hiring managers, and fellow SDETs evaluating David (Davo)'s testing expertise. Visitors arrive with a professional intent: assessing technical capability, evaluating cultural fit, or discovering testing frameworks.

## Product Purpose

Showcase David's testing expertise through a memorable Metal Gear Solid 1 Codec-themed portfolio. The site functions as an interactive CV, a testing framework showcase (Tactical Espionage Testing Suite), and a personal brand surface. Success means a visitor leaves remembering "the SDET with the codec portfolio" — distinctiveness is the metric.

## Brand Personality

Tactical, precise, nostalgic. The voice of a codec operator: direct, technical, no filler. Three words: **military-grade, terminal, committed**.

## Anti-references

- Generic dark portfolio templates (slate-900 + amber accents = invisible)
- Corporate blandness (stock photos, gradient heroes, "I'm passionate about quality")
- Editorial-magazine aesthetic (display serif + italic headlines + broadsheet grid) — wrong register entirely
- "AI slop" — any design that looks like it was generated, not crafted

## Design Principles

1. **The codec IS the product** — Every design decision reinforces the MGS1 Codec metaphor. Green phosphor, scanlines, tactical UI. Not decoration; identity.
2. **Technical credibility through aesthetic commitment** — A testing framework deserves a testing-grade interface. Precision in design signals precision in code.
3. **Minimal interface, maximum impact** — Two buttons on the landing page. One framework per detail page. Restraint is tactical, not lazy.
4. **Show, don't tell** — Framework capabilities demonstrated through the interface itself, not described in paragraphs.
5. **$0 ambition** — Prove that open-source tools and creative design can rival paid solutions. The constraint is the feature.

## Accessibility & Inclusion

- WCAG 2.1 AA minimum (WCAG AAA contrast achieved: green #00FF41 on black #0D0208 = 10.3:1)
- `prefers-reduced-motion` support: scanlines, flicker, and waveform animations disabled
- Semantic HTML throughout (headings, landmarks, ARIA labels)
- Keyboard navigation for all interactive elements
- Screen reader friendly: meaningful alt text, aria-labels on icon-only buttons
